## 🎯 Sobre o Projeto

O SUNSET é um sistema web desenvolvido para facilitar o agendamento de quadras para práticas de **Beach Tennis** e **Vôlei**. Com uma interface simples e responsiva, você pode:

- ✅ Criar sua conta em poucos segundos
- ✅ Agendar horários disponíveis em quadras específicas
- ✅ Receber confirmação instantânea do seu agendamento
- ✅ Gerenciar seus agendamentos facilmente

**Público-alvo:** Clientes interessados em praticar Beach Tennis ou Vôlei de forma organizada e sem complicações.

## 🚀 Como Começar

### 1. Acessar a Plataforma
Abra o arquivo `PIM_ATUALIZADO.HTML` no seu navegador web preferido (Chrome, Firefox, Safari, Edge).

### 2. Primeira Visita
Você verá a tela de **Login**. Se não possui uma conta, clique em **"Criar conta"**.

### 3. Criar Sua Conta
Preencha os dados solicitados:
- **Nome Completo**
- **E-mail**
- **Usuário**
- **Senha** (mínimo recomendado: 6 caracteres)
- **Confirmar Senha**

Clique em **"Cadastrar"** e sua conta está pronta!

### 4. Fazer Login
Após criar sua conta, volte à tela de login e insira:
- Seu **Usuário**
- Sua **Senha**

Clique em **"Entrar"** para acessar a plataforma.

## 📖 Guia do Usuário

### Agendando um Horário

1. **Faça Login** com suas credenciais
2. **Você será redirecionado** para a tela de agendamento
3. **Preencha os dados:**
   - **Esporte:** Selecione entre Beach Tennis ou Vôlei
   - **Quadra:** Escolha entre as quadras disponíveis (1, 2 ou 3)
   - **Data:** Selecione o dia desejado usando o calendário
   - **Horário:** Escolha um horário entre 08:00 e 20:00

4. **Clique em "Confirmar Agendamento"**
5. **Veja a confirmação** com todos os seus dados
6. **Pronto!** Seu agendamento foi salvo

### Cancelando um Agendamento

- Faça login na sua conta
- Vá até a tela de confirmação
- Você poderá voltar ao login para fazer novos agendamentos
- ⚠️ *Nota: Para cancelar agendamentos existentes, entre em contato com um administrador*

## 🔧 Painel Administrativo

### Acessando o Painel Admin

1. **Na tela de Login**, marque a checkbox **"Sou administrador"**
2. **Insira suas credenciais** de administrador
3. **Clique em "Entrar"**
4. **Você será redirecionado** para o Painel Administrativo

### Funcionalidades do Admin

#### 📊 Visualizar Agendamentos
O painel exibe uma tabela com todos os agendamentos registrados, mostrando:
- Nome do Cliente
- Esporte agendado
- Quadra reservada
- Data do agendamento
- Horário

#### 🔍 Filtrar Agendamentos

**Por Nome:**
- Use o campo **"Pesquisar por nome..."**
- Digite o nome do cliente
- Clique em **"Filtrar"**

**Por Data:**
- Selecione uma data no campo **"Data"**
- Clique em **"Filtrar"**

**Combinado:**
- Preencha nome E data
- Clique em **"Filtrar"** para resultados específicos

#### 🗑️ Excluir Agendamento
- Localize o agendamento na tabela
- Clique no botão **"Excluir"** na coluna Ações
- O agendamento será removido imediatamente

#### 🔄 Limpar Filtros
- Deixe os campos vazios
- Clique em **"Filtrar"** para ver todos os agendamentos

## 📝 Dúvidas Frequentes (FAQ)

### ❓ Como faço para criar uma conta?
Clique em "Criar conta" na tela inicial, preencha seus dados e clique em "Cadastrar".

### ❓ Esqueci minha senha. O que faço?
Entre em contato com um administrador. Atualmente, não há função de recuperação de senha automática.

### ❓ Posso agendar múltiplas quadras no mesmo horário?
Sim! Você pode fazer quantos agendamentos desejar, desde que preencha o formulário completo para cada um.

### ❓ Como cancelar meu agendamento?
Você pode solicitar o cancelamento a um administrador, que removerá seu agendamento do sistema.

### ❓ Os dados dos meus agendamentos são salvos?
Sim! Seus agendamentos são salvos localmente no seu navegador usando localStorage. Se limpar o histórico ou cookies, os dados podem ser perdidos.

### ❓ A plataforma funciona em celular?
Sim! O SUNSET é totalmente responsivo e funciona perfeitamente em smartphones e tablets.

### ❓ Há limite de agendamentos por usuário?
Não há limite definido. Você pode agendar quantos horários desejar.

## 🛠️ Resolução de Problemas

### Problema: "Não consigo fazer login"
**Solução:**
- Verifique se usuário e senha estão corretos
- Certifique-se de que criou sua conta anteriormente
- Limpe o cache do navegador e tente novamente

### Problema: "Meus agendamentos desapareceram"
**Solução:**
- Se limpou o histórico/cookies, os dados serão perdidos
- Tente restaurar dados do navegador
- Contate um administrador para recriar os agendamentos

### Problema: "A página não carrega corretamente"
**Solução:**
- Recarregue a página (F5 ou Ctrl+R)
- Verifique sua conexão de internet
- Tente em outro navegador
- Limpe o cache e cookies

### Problema: "VLibras não está funcionando"
**Solução:**
- Verifique sua conexão de internet
- Recarregue a página
- O serviço VLibras pode estar indisponível temporariamente
